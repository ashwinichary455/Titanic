# Titanic Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand survival patterns of passengers.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- PassengerId
- Name
- Pclass
- Sex
- Age
- SibSp
- Parch
- Ticket
- Fare
- Cabin
- Embarked
- Survived

## Data Cleaning
- Handled missing values
- Checked duplicates
- Performed data preprocessing

## Visualizations
- Histogram
- Countplot
- Boxplot
- Heatmap

## Key Insights
- Female passengers had higher survival rates
- First-class passengers had better survival chances
- Age and fare impacted survival trends
